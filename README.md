# 个人博客系统

**技术架构**

- 前端：React 18、TypeScript、Vite、Axios、Redux、Echarts、Antd UI、Scss、Motion React、React Form Hooks

- 后端：Nodejs、Koa2、Koa-router、Redis、Sequelize、MySQL

**实现功能**
- 前台：主页 + 搜索页 + 我的收藏页 + 个人信息修改页 + 文章详情页 + 登录注册页
- 后台：仪表盘 + 用户管理 + 文章管理 + 标签管理 + 分类管理
